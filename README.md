## 📋 О проекте

Single Page Application для автоматизации записи клиентов в салон красоты. Три роли пользователей:

- **Клиент** — запись на услуги через многошаговую форму
- **Специалист** — просмотр своего расписания
- **Менеджер-администратор** — полный контроль записей (CRUD)
