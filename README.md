# 🏥 bookme24.by — Система онлайн-записи для салона красоты

**BookMe24** — это современная система онлайн-записи для салонов красоты «Здоровье и красота». Приложение позволяет клиентам записываться на услуги 24/7, а администраторам — эффективно управлять расписанием, услугами и специалистами.

## 🌟 Особенности

### Для клиентов
- ✅ **Онлайн-запись** — запишитесь за 5 простых шагов (менее 2 минут)
- ✅ **Каталог услуг** — подробное описание услуг с фильтрацией и сортировкой
- ✅ **Выбор специалиста** — просмотр профилей мастеров и их услуг
- ✅ **Личный кабинет** — управление своими записями, история посещений
- ✅ **Избранное** — быстрый доступ к любимым услугам и мастерам
- ✅ **Бонусная система** — накопление баллов за посещения
- ✅ **Две языковые версии** — русский и английский языки
- ✅ **Тёмная/светлая тема** — комфорт в любое время суток

### Для администраторов
- ✅ **Управление записями** — просмотр, редактирование, отмена записей
- ✅ **Управление услугами** — добавление, редактирование, удаление услуг
- ✅ **Управление специалистами** — назначение услуг мастерам
- ✅ **Статистика** — отслеживание выручки, количества записей
- ✅ **Фильтры и поиск** — быстрый поиск по клиентам, датам, статусам
- ✅ **Валидация данных** — защита от некорректного ввода

## 🚀 Технологии

### Frontend
- **React 18.2+** — современная библиотека для создания UI
- **React Router v6** — клиентский роутинг
- **Context API** — управление состоянием (тема, язык)
- **Custom Hooks** — переиспользуемая логика
- **CSS3** — стилизация с CSS-переменными и BEM
- **Lucide React** — иконки

### Хранение данных
- **localStorage** — персистентность данных на клиенте
- **JSON-файлы** — начальный каталог услуг и специалистов

### Инструменты
- **Create React App** — сборка проекта
- **ESLint** — линтинг кода
- **Prettier** — форматирование кода

## 📦 Установка

### Требования
- Node.js >= 16.x
- npm >= 8.x

### Шаги установки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/igorao2802-dev/bookme24.git
cd bookme24
```

2. **Установите зависимости:**
```bash
npm install
```

3. **Запустите проект:**
```bash
npm start
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

4. **Сборка для production:**
```bash
npm run build
```

### Константы приложения

Основные константы находятся в `src/utils/constants.js`:

- **BOOKING_STATUS** — статусы записей (pending, confirmed, inProgress, completed, cancelled)
- **SERVICE_CATEGORIES** — категории услуг (hair, nails, massage, cosmetology, spa)
- **STORAGE_KEYS** — ключи для localStorage
- **BUSINESS_CONFIG** — бизнес-правила (время работы, буфер между записями и т.д.)
- **PRICE_LIMITS** — ограничения цен (min, max, step)
- **RATE_LIMITS** — ограничения для защиты от спама

##  Функциональность

### Многошаговая запись (5 шагов)

1. **Выбор услуги** — поиск, фильтрация по категории, цене, рейтингу
2. **Выбор специалиста** — отображение только мастеров, оказывающих выбранную услугу
3. **Дата и время** — выбор свободных окон с учётом расписания мастера
4. **Контактные данные** — валидация телефона (РБ), email, комментария
5. **Подтверждение** — проверка всех данных перед созданием записи

### Каталог услуг и специалистов

**Фильтры:**
- Категория услуги (Все, Волосы, Ногти, Массаж, Косметология, SPA)
- Диапазон цен (от 0 до 500 BYN)
- Рейтинг (Любой, 4+, 4.5+, 4.8+)

**Сортировка:**
- По популярности
- По цене (возрастание/убывание)
- По рейтингу
- По алфавиту
- По категории

### Админ-панель

**Управление записями:**
- Просмотр всех записей с пагинацией
- Фильтрация по статусу, специалисту, дате
- Редактирование записей
- Отмена записей с подтверждением

**Управление услугами:**
- Добавление новых услуг
- Редактирование существующих
- Удаление (только кастомных услуг)
- Назначение специалистов на услуги
- Валидация полей (название, категория, описание, длительность, цена)

**Управление специалистами:**
- Добавление новых мастеров
- Редактирование данных
- Удаление (только кастомных)
- Назначение услуг специалистам

### Личный кабинет

**Разделы:**
- Профиль (ФИО, телефон, email)
- Статистика (всего записей, предстоящие, подтверждённые, отменённые)
- История записей (все, предстоящие, завершённые, отменённые)
- Избранное (услуги и специалисты)
- Настройки (уведомления, способ связи)
- Бонусный счёт (баланс, уровень, история начислений)

## 🎨 Темы и локализация

### Темы

Приложение поддерживает две темы:
- **Светлая** — по умолчанию
- **Тёмная** — переключается через кнопку в хедере

Все цвета определены в CSS-переменных (`src/styles/variables.css`) для лёгкой кастомизации.

### Локализация

Поддерживаются два языка:
- **Русский (ru-RU)** — по умолчанию
- **Английский (en-US)**

Переключение языка доступно через кнопку в хедере. Все тексты вынесены в файлы локализации (`src/i18n/`).

## 🔒 Безопасность

### Rate Limiting

Защита от многократных кликов:
- ≤ 3 клика за 5 секунд (короткое окно)
- ≤ 10 кликов за 30 секунд (длинное окно)
- Блокировка на 30 секунд при превышении лимита

### Валидация

**Телефон (РБ):**
- Формат: +375XXXXXXXXX (12 цифр)
- Разрешённые коды операторов: 25, 29, 33, 44

**Email:**
- Валидация по RFC 5321
- Максимальная длина: 254 символа

**Цены:**
- Только целые числа
- Диапазон: 0 — 10000 BYN

**Длительность:**
- Минимум: 15 минут
- Максимум: 480 минут (8 часов)
